## Train Generator JS

A simple JavaScript-based generator function that simulates a train journey on the Metro North Railroad. It yields the next station on the journey with each button click, until it reaches Grand Central Station.

## Features

Simulates a train journey using JavaScript generator functions.
Displays the next station with every button click.
Automatically disables the button and shows a message once the final station, Grand Central, is reached.

## How to Use

Clone or download the repository.
bash
Copy code
git clone https://github.com/marna4255/train-generator-js.git
Open index.html in your browser.

Click the "Next Station" button to see the next station in the journey.
